Tacit is a CSS framework for dummies, who want their web services look attractive but have almost zero skills in graphic design, just like myself. This is how your website will look if just add this code to your HTML: 

<!DOCTYPE html>
  <html>
    <head>
      <link rel="stylesheet" type="text/css" href="//yegor256.github.io/tacit/tacit-0.1.1.min.css"/>
    </head>
  </html>

You should not use any CSS classes to make your HTML code look nice. Just make sure it is HTML5 compliant and you'll be fine. Again, no classes, just standard HTML elements, like <table> and <input>.
